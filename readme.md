# BOOKFAVE

You can see the example in [https://bookfave.netlify.com/](https://bookfave.netlify.com/)

### How to use

#### Install

```
yarn install
```

or

```
npm install
```

#### Run project

```
yarn start
```

It will run a server in [http://localhost:3000](http://localhost:3000)

#### Run tests

```
yarn test
```
